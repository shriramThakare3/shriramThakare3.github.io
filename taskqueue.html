Task Queue API
1.0.0
Async background job processing. Submit tasks via REST, workers pick them up from PostgreSQL using SELECT FOR UPDATE SKIP LOCKED. Supports priority scheduling, retries, and horizontal worker scaling.
Base URL: http://localhost/ · Run locally with Docker Compose · docker compose up --build
tasks — submit & manage background jobs▼
POST /tasks Submit a new task
Request Body
{ "name": "add", // add | sleep | reverse | fibonacci | word_count | file_write "input_data": { "a": 21, "b": 21 }, "priority": 5, // 0-100, higher = processed first "max_retries": 3 }
Responses
201Task created. Returns full task object with id and status=pending.
400Unknown task name or invalid input.
422Validation error.
