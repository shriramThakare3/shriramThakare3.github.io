# shriramthakare3.github.io

Personal portfolio and live demo site for my backend projects.

## Live Site
**[shriramthakare3.github.io](https://shriramthakare3.github.io)**

## Projects Demonstrated
- **Task Queue System** — Mini Celery Clone (FastAPI + PostgreSQL + Workers)
- **Mini Cloud Storage** — Google Drive-style file API
- **Authentication Server** — JWT + bcrypt + RBAC

## How to Connect Live Demos
1. Deploy a project to Railway (see project READMEs)
2. Visit the portfolio site
3. Paste your Railway URL into the **API BASE URL** field
4. All three demos connect live to your running backend

## Stack
- Pure HTML/CSS/JS — no framework, no build step
- Hosted on GitHub Pages (free, zero config)
- Connects to Railway-hosted backends via fetch API
